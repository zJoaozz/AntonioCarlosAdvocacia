## 📋 Sobre o Projeto

O site institucional de **Antonio Carlos da Costa** (OAB/PI 1.977) apresenta o escritório, suas áreas de atuação e diferenciais, com foco em conversão via WhatsApp. Desenvolvido com identidade visual premium em verde escuro e dourado, tipografia serifada elegante e microinterações refinadas.

## ✨ Funcionalidades

### 🌐 Site Público
- Hero com brasão, assinatura tipográfica e call-to-actions
- Seção institucional com contadores animados (OAB, especialidades, anos de experiência)
- 3 cards de Áreas de Atuação (Trabalhista, Previdenciário, Imobiliário e Patrimonial)
- 6 cards de Diferenciais do escritório
- Mapa do Google integrado com a localização do escritório
- Botão flutuante de WhatsApp com pulse animation
- Banner de consentimento LGPD
- Menu hambúrguer para mobile
- Cursor customizado dourado (desktop)
- Animações de *reveal* ao rolar a página
- Design 100% responsivo (mobile-first)

## ⚙️ Como Rodar Localmente

### Pré-requisitos
- Qualquer servidor estático (Live Server, `npx serve`, etc.)

### 1. Clone o repositório
```bash
git clone https://github.com/TriCod3x/antonio-carlos-costa.git
cd antonio-carlos-costa
```

### 2. Abra com um servidor local
Abra `index.html` com o Live Server (VS Code) ou rode:
```bash
npx serve .
```

Acesse `http://localhost:5500` (ou a porta indicada pelo terminal).

---

## ☁️ Deploy

O projeto está configurado para deploy estático na **Vercel**:

```bash
npx vercel
```

Basta seguir o fluxo de login e confirmação — o site fica no ar em menos de um minuto.
